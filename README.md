# Application Mobile – Gestion des Contacts avec Firebase

## Description
Cette application mobile Android permet de gérer une liste de contacts en utilisant **Firebase ** comme base de données en ligne.  
L’utilisateur peut ajouter, modifier, supprimer et afficher des contacts en temps réel.

---

## Objectifs du projet
- Découvrir le développement mobile Android
- Utiliser Firebase Firestore pour le stockage des données
- Créer une interface simple et intuitive
- Implémenter les opérations CRUD (Create, Read, Update, Delete)

---

## Technologies utilisées
- **Android Studio**
- **Java**
- **XML**
- **Firebase Firestore**
- **Git & GitHub**
